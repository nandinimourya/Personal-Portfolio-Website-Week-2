Projects & Experience

One of my major projects is CareLink — a full-stack MERN web application that connects NGOs, volunteers, and donors on a single platform. I built this collaboratively with my team, and it features role-based access for Admins, NGOs, and Users, along with fundraising, event management, volunteer registration, NGO verification, and an in-app notification system. Another project I have worked on is Wanderlust, a travel listing web application. These projects have given me real experience in designing, building, and deploying complete web solutions.
